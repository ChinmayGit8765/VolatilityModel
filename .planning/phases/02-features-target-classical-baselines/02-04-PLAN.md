<formulas>
Exact forms (from 02-RESEARCH Code Examples — directive, not to be inlined as code blocks):
- Parkinson variance: ln(high/low)**2 / (4 * ln(2)).
- Garman-Klass variance: 0.5*ln(high/low)**2 - (2*ln(2)-1)*ln(close/open)**2. (Apply to equities too; note in feature docstring that it omits the overnight gap for equities — Open Question #4.)
- Realized variance over window w: log_returns.pow(2).rolling(w).mean(), integer w in {5,10,22,66}.
- Vol-of-vol: rolling std of a realized-vol series (e.g. rolling std of rv_22 over a window) — document the chosen window.
- Rolling skew/kurt: log_returns.rolling(w).skew() / .kurt() over a documented window.
- Lagged vol: realized-vol series shifted by k (e.g. rv_22.shift(1)).
- Cross-asset as-of join: pd.merge_asof(direction="backward", tolerance=pd.Timedelta("3D")); both frames sorted ascending by the "date" key BEFORE the call; values beyond 3 calendar days become NaN.
- Calendar features: dayofweek (0=Mon..4=Fri), month (1-12), is_monday/is_friday flags; for equities a session/overnight flag (processed equity data has no weekend rows).
</formulas>
