<arch_api_facts>
Verified in 02-RESEARCH Pattern 1 (cite bashtage.github.io/arch):
- `arch_model(scaled, mean="Zero", vol="GARCH", p=1, q=1, rescale=False)` then `.fit(disp="off")`.
- `res.convergence_flag == 0` means success — assert it.
- `res.params["alpha[1]"]`, `res.params["beta[1]"]` — assert `alpha + beta < 1.0` (stationarity).
- `res.forecast(horizon=h, reindex=False).variance["h.1"...]` — multi-step forecast for the test window.
- `res.conditional_volatility` returns STD DEV — square for variance.
- GARCH_SCALE = 100.0 module constant. Forecast variance is in (100*r)^2 units → divide by GARCH_SCALE**2 exactly once.
- Refit every 21 steps (monthly); from each refit emit forecasts for all `step` test days via forecast(horizon=step). Fallback on convergence failure: previous window's params, then EWMA; log + count fallbacks (Open Question #1).
</arch_api_facts>
